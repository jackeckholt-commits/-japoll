About the Project

A simple, transparent approach to comparing national polling averages.

Purpose

This site acts as an aggregator of aggregators. Instead of relying on one polling average, it compares several polling averages to get a broader view of the political environment.

Sources

The tracker compares national averages from sources such as VoteHub, Silver Bulletin, The New York Times, CNN, Race to WH, and FiftyPlusOne.

Average of averages

The current average gives each available source equal weight. Sources without an available average are listed, but they are not included in the calculation until a usable value is available.

Current calculations

The generic ballot average currently uses VoteHub, Silver Bulletin, CNN, Race to WH, and FiftyPlusOne. The New York Times is listed as unavailable for the generic ballot because no usable national generic ballot average is available.

The Trump approval average currently uses The New York Times, Silver Bulletin, CNN, VoteHub, Race to WH, and FiftyPlusOne.

Algorithm adjustment

The generic ballot page includes an optional adjustment based on my algorithm. This can be set as a Democratic or Republican point adjustment, such as D+1.6 or R+1.6, and it applies only to the displayed generic ballot average and the generic ballot trend graph.

The raw source cards still show the values reported by each aggregator. The adjustment can be edited in data/adjustments.json.
